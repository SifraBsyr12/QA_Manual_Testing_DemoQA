# Bug Report - BUG001

**Bug ID:** BUG001  

**Title:** Show Password (eye) icon missing on Login Page  

**Module / Feature:** Login Page  

**Environment:**  
- Browser: Chrome  
- OS: Windows 10  
- URL: https://demoqa.com/login  

**Severity:** Medium  
**Priority:** Medium  

---

## Description
The Show Password (eye) icon is not displayed on the login page, so users cannot toggle password visibility.

---

## Steps to Reproduce
1. Navigate to Login page  
2. Enter password field  
3. Look for Show Password (eye) icon  

---

## Expected Result
Password should be visible in plain text when clicking the eye icon.

---

## Actual Result
Show Password (eye) icon is not available on the login page.
